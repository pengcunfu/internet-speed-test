# 网速测试工具 - Internet Speed Test Tool

基于Python和PySide6开发的现代化网络速度测试工具，提供直观的图形用户界面。

## ✨ 功能特性

- 🚀 **下载速度测试** - 测试网络下载速度
- 📤 **上传速度测试** - 测试网络上传速度  
- 🔄 **完整速度测试** - 同时测试下载和上传速度
- 🏓 **Ping测试** - 测试网络延迟
- 🌐 **IP信息查询** - 查看本机或外部IP的详细信息
- 📋 **结果复制** - 一键复制测试结果到剪贴板

## 🖼️ 界面预览

现代化的图形用户界面，支持：
- 直观的按钮布局
- 实时进度显示
- 清晰的结果展示
- 响应式设计

## 📋 系统要求

- Python 3.7+
- Windows 10/11, macOS 10.14+, 或 Linux

## 🚀 快速开始

### 方法一：使用启动脚本（推荐）

1. 克隆或下载项目到本地
2. 运行启动脚本：
   ```bash
   python run.py
   ```
   启动脚本会自动检查依赖包并启动程序。

### 方法二：手动安装

1. 安装依赖包：
   ```bash
   pip install -r requirements.txt
   ```

2. 运行程序：
   ```bash
   python main.py
   ```

## 📦 依赖包

- **PySide6** >= 6.4.0 - 现代化GUI框架
- **speedtest-cli** >= 2.1.3 - 网速测试核心库
- **requests** >= 2.28.0 - HTTP请求库
- **pyperclip** >= 1.8.2 - 剪贴板操作库

## 🔧 使用说明

### 主要功能

1. **下载速度测试**
   - 点击"下载速度测试"按钮
   - 等待测试完成
   - 查看下载速度结果（Mbps）

2. **上传速度测试**
   - 点击"上传速度测试"按钮
   - 等待测试完成
   - 查看上传速度结果（Mbps）

3. **完整速度测试**
   - 点击"完整速度测试"按钮
   - 依次测试下载和上传速度
   - 获得完整的网速报告

4. **Ping测试**
   - 点击"Ping测试"按钮
   - 查看网络延迟（ms）

5. **IP信息查询**
   - 点击"IP信息"按钮
   - 选择查询类型：
     - 获取我的IP
     - 获取我的IP详细信息
     - 查询外部IP信息

### 注意事项

- 测试过程需要网络连接，请确保网络正常
- 测试时间取决于网络状况，通常需要10-30秒
- 建议在测试期间不要进行其他大流量网络活动
- 可以复制测试结果保存或分享

## 🛠️ 技术架构

- **GUI框架**: PySide6 (Qt6的Python绑定)
- **网速测试**: speedtest-cli库
- **多线程**: QThread实现非阻塞UI
- **网络请求**: requests库
- **数据格式**: JSON格式的API响应
